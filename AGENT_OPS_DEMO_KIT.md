## Recording Checklist

Before recording:
- Close unrelated tabs.
- Use a clean browser profile if possible.
- Zoom browser to 100% or 110%.
- Hide bookmarks if cluttered.
- Put fake data in place before starting.
- Make sure no private email, token, credential, dashboard, or client info is visible.
- Open all tabs in recording order.
- Practice once without recording.

During recording:
- Move slowly.
- Do not scroll too fast.
- Keep cursor movement intentional.
- Pause 1-2 seconds on important output screens.
- Keep each video under 90 seconds.

After recording:
- Trim dead air at start and end.
- Add simple title text if your tool supports it.
- Export 1080p MP4.
- Name files:
  - lead-follow-up-agent-demo.mp4
  - client-intake-agent-demo.mp4
  - proposal-drafting-agent-demo.mp4
